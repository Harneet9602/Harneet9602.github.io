NLP-Powered Mental Health Status Detection

Built an NLP pipeline to classify 51,000+ social media posts into 7 distinct mental health categories. Addressed severe class imbalance using stratified sampling and metric-driven evaluation.

Key Innovation: Benchmarked MentalBERT, BioBERT, and PubMedBERT; selected MentalBERT based on superior minority-class performance (F1 = 0.84), especially for Personality Disorders.
